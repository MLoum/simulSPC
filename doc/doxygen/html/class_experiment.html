Detailed Description

Main structure of the programm.

This class gather all the instance of the parameters (time_step_, space_step_, OpticalSetup, solvent, PSF, CEF, ...) and the list of the particle particle_list. The main function is "tick" who simulate the experience for one tick of the main clock. Some members are for convenience (like in python)
